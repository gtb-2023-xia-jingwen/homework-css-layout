![todo-list](mockup.jpg)


## 提示

此问题需要修改 _public/styles_ 目录下的 index.css 文件，可以适当修改 index.html 文档结构

字体，颜色，宽高，间距大概一致即可，不必精确到像素级别

可以尝试使用不同的布局方式实现相同的效果

## 下载依赖

```bash
$ npm install
```

## 运行网页

请执行如下代码运行网站并打开页面：

```bash
$ npm start
```

之后在浏览器中访问：http://localhost:8080
